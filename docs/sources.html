How to Locate Native RSS Feeds from External Media

Global mainstream tech media (BBC Tech, Reuters Tech, Ars Technica) provide native RSS feeds on their official websites.

Workflow: Open media homepage via browser, locate "RSS Feed" entry at page footer, copy raw RSS URL and manually add inside Swiftfeed App.
